Building a Tesseract: Dimension by Dimension

Discover how each dimension builds on the last — from a point all the way to the fourth dimension.

From Point to Line (0D → 1D)

Let's start with the simplest thing: a single point. A point has no size — it's just a location. It exists in zero dimensions.

But what happens when we move that point? Drag the teal point to the right and watch what gets created.

Drag the teal point horizontally to create a line segment.

Amazing! The line segment you created is the path the point traced as it moved. A line has one dimension — length. Every point along that line is a position the moving point occupied.

The pattern begins: Moving a 0D object (point) through space creates a 1D object (line).

Check your understanding: If you drag the point 7 units to the right, how long is the line segment? units

From Line to Square (1D → 2D)

Now we have a line segment. What if we apply the same idea — but move the entire line in a new direction?

Use the growth slider to extend the line perpendicular to itself. Watch the line "grow" a second dimension.

Growth: 0 units

Drag the slider to grow the line into a square. Try moving backward too!

The line didn't just stretch — every point on the line simultaneously moved perpendicular to itself, sweeping out a square. The original line (shown dashed) became one edge, and its journey created the area.

The pattern continues: Moving a 1D object (line) perpendicular to itself creates a 2D object (square).

Check your understanding: A line segment of length 5, grown 5 units perpendicular, creates a square. What's its area? square units

From Square to Cube (2D → 3D)

We have a square. Now let's apply the same pattern again: move the entire square perpendicular to itself — into the third dimension.

Click and drag the teal arrow to pull the square into 3D space.

Drag the teal circle diagonally up-left to pull the square into 3D.

The square became the front face of a cube. Its copy (shown in gray) became the back face. The four corners of the original square traced out the four connecting edges.

The pattern holds: Moving a 2D object (square) perpendicular to itself creates a 3D object (cube).

Check your understanding: A square has 4 vertices. When we copy it and connect corresponding vertices, how many vertices does the cube have? vertices

From Cube to Tesseract (3D → 4D)

Here's where it gets mind-bending. We can't see a fourth perpendicular direction — but mathematically, we can follow the exact same pattern.

To understand what a tesseract (4D hypercube) is, let's compare how we slice through dimensions. Use the slice slider to cut through both shapes.

Slicing a 4D Tesseract
Slice position: 50%

Move the slice through each shape. Notice: cube slices are squares, tesseract slices are cubes!

Just as a cube is made of infinitely many square slices stacked in the third dimension, a tesseract is made of infinitely many cube slices stacked in the fourth dimension.

The pattern completes: Moving a 3D object (cube) perpendicular to itself (into 4D) creates a 4D object (tesseract).

Check your understanding: A cube has 8 vertices. Following the doubling pattern, how many vertices does a tesseract have? vertices

Seeing the Pattern

Let's step back and see the beautiful pattern you've discovered. For each shape below, draw an arrow in the direction you think is "perpendicular to itself" — the direction you'd move it to create the next dimension.

Drag from each shape's center to draw your predicted perpendicular direction, then press Sweep.

The universal rule: To go up one dimension, take your shape and move it perpendicular to all its existing dimensions, then connect corresponding points.

0D → 1D
Point → Line
1 → 2
1D → 2D
Line → Square
2 → 4
2D → 3D
Square → Cube
4 → 8
3D → 4D
Cube → Tesseract
8 → 16

Each dimension doubles the vertex count — because you're making a copy and connecting every vertex to its twin. You've just understood a tesseract not as something mystical, but as the natural next step in a pattern that starts with a simple point.
